Raving Software

Raving Software (View Site )

Raving Software is a web design company I worked at from 2017 - 2018. Part of my responsibilities were planning, designing, and coding websites for our clients and for the company's website. All the other websites listed in the "Raving Software" section are either in-house or client websites.

Marie Chantal's Bakery (View Site )

For this small bakery, we developed a theme through establishing a color scheme, font choices, and repeated elements. In addition, we improved her online ordering and reviewing processes, and trained the client on how to update her photo galleries and the image slide, and how to manage reviews submitted via the website.

The Delightful Cupcake (View Site )

This is a website I built on my own as a theme example for Raving Software. We wanted to showcase our abilities and give clients an idea of what we could do for their website. Having examples of other websites helps the client to figure out what they do and don't like, and what might work best for their business needs.

Silver Star Service Center, Inc. (View Site )

We built the website to assist in the business's workflow by incorporating service appointment scheduling, a vehicle sales section, a customer rating capaiblity and customer contact forms, all of which can be accessed by the client through the Wordpress CMS. In addition to improving workflow, we created copy and artwork for the website to make it visually appealing and informative for customers.

Anne Arundel Community College

For my Internet Applications Development degree, I took courses in HTML and CSS, JavaScript, jQuery, PHP and mySQL, and Java. I have selected a few examples of my work to show where I started and my progress. I took my first web design course in 2013 at the University of Missouri, but had not done any web work since.

Casa Della Nonna (View PDF of Site )

This is a single-page website for a local restaurant's menu. The site is mobilized, though I would go back and fix the H1 headline size as it takes up too much space vertically when the browser window is small. Other than that, I'm happy with how this website turned out.

Order of the Swan (View PDF of Site )

This website was created for a fake band. All the creative decisions were left up to me, except for the band name and three images that were all pulled from a random search on Wikipedia. I had a lot of fun working on this website and creating a tone and content for my fake band. The site is not mobilized, but I would go back and use Bootstrap and media queries to make it mobile-friendly.

Type and Positioning Web Page (View Page )

For this web page, I had to choose a typeface and design a digital typeface poster than conveyed it's personality. I chose the Google Font of Alegreya and used shapes and text, per the assignment requirements, to create a page of an open book. I chose to create a book because the font was designed for works of literature.

Favorite Book with JavaScript Scrolling (View PDF of Site )

This website was an introduction to JavaScript. I had to create a web page for my favorite book and its author. I chose to do "Mansfield Park" by Jane Austen because I figured it was not one that would be done by a lot of people. I like how this page turned out, but there are a few positioning elements that I would work on.

Explore: Get Out and Go (View PDF of Site )

As a final project, I designed a multi-page, mobilized website. I chose to build a website that would focus on providing people with activities to do in the Baltimore-Annapolis-D.C. area. Certain elements, such as tables, lists, and contact forms, had to be utilized in the website. Besides the requirements, I had complete control over the creative choices. I used my own photos or created graphics for the website where I could, and used Creative Common licensed photos for the rest. The background is actually a repeated image of the blue color that I texturized in PhotoShop. There's a back-up background of the solid color blue in case the images can't load.

Friends with Paws (View PDF of Site )

Another class also required a multi-page, mobilized website for a final assignment. Instead of reusing my "Explore" website, I decided to do another one from scratch. While it does mobilize, it doesn't execute as well as I would like. The structure I used for the "Explore" website worked much better than what I did for this one. One of my current classes focuses on responsive design so I'm hoping to improve these skills throughout the semester. Note that the blog has been deleted, but once linked to a Wordpress blog for the Friends with Paws veterinary service.
